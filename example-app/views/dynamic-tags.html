Dynamic tags

You can update meta tags programmatically by injecting ngMeta into your controller 

    angular.module('yourApp')
    .controller('SomeController', function($scope, ngMeta) {
      
      $scope.someMethod = function(tagName, tagValue) {
        //Some method called, set the tag
        ngMeta.setTag(tagName, tagValue);
      };
      
      $scope.someOtherMethod = function(articleName) {
        //Change title based on selected article
        ngMeta.setTitle(articleName);
      };
    });

Set ngMeta.title as the value of the og:title meta tag 

      Title (ngMeta.title):
      {{ngMeta.title}}

Example: 

      <meta property="og:title" content="{{ngMeta.title}}" />

Similarly, you can set and refer to arbitrary meta tags 

      Author (ngMeta.author): {{ngMeta.author}}

Example: 

      <meta property="og:author" content="{{ngMeta.author}}" />
